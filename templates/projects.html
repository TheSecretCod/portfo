Mis Proyectos

Una selección de desarrollos técnicos enfocados en innovación y eficiencia.

OCR-WEB: Digitalización Inteligente

Una aplicación web avanzada que transforma trazos manuales y dibujos en texto digital editable. Desarrollada con Python y Flask, integra potentes algoritmos de OCR (Reconocimiento Óptico de Caracteres) para ofrecer una experiencia de digitalización fluida y precisa.

Conversor de Divisas Global

Solución robusta desarrollada en Java para la conversión de divisas en tiempo real. Este sistema gestiona múltiples tipos de cambio y ofrece una interfaz eficiente para realizar cálculos financieros precisos, demostrando un manejo sólido de la lógica de negocio y estructuras de datos.

Encrypted-Challenge: Seguridad de Texto

Implementación de un sistema de encriptación y desencriptación de texto diseñado para proteger la información confidencial. Construido utilizando HTML, CSS y JavaScript (Logica en Java/JS), este proyecto destaca por su enfoque en la seguridad de datos y la manipulación de cadenas para criptografía básica.

Ver código

Aleatory Game: Lógica Probabilística

Un desafío interactivo de adivinanza numérica que pone a prueba la lógica y la intuición del usuario. Desarrollado con Java, HTML y CSS, implementa algoritmos de generación de números aleatorios y control de flujo para gestionar intentos y puntuaciones en un entorno web dinámico.
